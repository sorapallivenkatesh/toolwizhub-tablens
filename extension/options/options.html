TabLens settings

Features

Turn each feature on or off. The three page features need permission to run on the sites you visit — you'll be asked once.

  • ⏱ Site usage timer

    Tracks how long you actively spend on each site.

  • 🔥 Streak dashboard

    Counts consecutive days you've browsed. Shown in the popup.

  • Page features — need access to the sites you visit

  • 📍 Scroll position memory

    Returns you to where you left off when you revisit a page.

  • 🗒 Sticky notes

    Pin draggable notes onto any page; they reappear on return.

  • 🔔 Page diff monitor

    Watches chosen pages and alerts you when their content changes.

Watched pages (page diff monitor)

URLs checked every 30 minutes for content changes.

    TabLens runs 100% locally — all timers, notes, and history stay in your browser.
